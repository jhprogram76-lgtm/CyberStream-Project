# 🛡️ CyberStream - SOC Analyst Dashboard Simulator

Este proyecto es un simulador de panel de control para un **SOC (Security Operations Center)** con una interfaz inspirada en plataformas de streaming modernas. El objetivo es demostrar habilidades en desarrollo frontend seguro, lógica de autenticación y visualización de activos críticos.

## 🚀 Características Técnicas

* **Autenticación Segura:** Sistema de login con validación de robustez de credenciales en el lado del cliente (JavaScript).
* **Gestión de Accesos:** Implementación de lógica para mitigar intentos de acceso no autorizados.
* **Interfaz Responsiva:** Diseño adaptado para entornos de monitorización profesional utilizando CSS Grid y Flexbox.
* **Simulación de Activos:** Visualización de nodos de red, bases de datos y firewalls en tiempo real.

## 🛠️ Tecnologías Utilizadas

* **HTML5 / CSS3:** Maquetación avanzada con efectos de cristalografía (Glassmorphism).
* **JavaScript (Vanilla):** Lógica de control de flujo y manejo de eventos de seguridad.
* **GitHub Pages:** Despliegue automático y gestión de entorno de producción.

## 📂 Mi Toolkit de Ciberseguridad

Este dashboard es el complemento visual de mis herramientas de auditoría en Python, las cuales incluyen:
1.  **Escáner de Puertos:** Detección de servicios activos y enumeración de banners.
2.  **Web Recon:** Localización de directorios sensibles y superficies de ataque.
3.  **Log Analyzer:** Scripts de detección de intrusiones en registros del sistema.
